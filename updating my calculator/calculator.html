Calculator

This is a demonstration of a calculator widget. This calculator features a transparent form that becomes solid when the mouse crosses it. The buttons light up when on mouse hover. When you click a button, the value of that button is added to the screen. Note: The buttons in red are currently not working. 

This project uses DOM control and dynamic HTML/CSS/JS design.
